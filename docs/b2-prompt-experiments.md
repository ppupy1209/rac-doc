# B2 프롬프트 실험 로그 — 환각률 vs 오거부율

RAG 답변 생성 프롬프트(`rag/RagService.java`)를 바꿔가며 `HallucinationEvalTest`로 측정한 기록.
지표: **환각률**(답 없는 20문항 중 지어낸 비율) / **오거부율**(답 있는 30문항 중 잘못 "모르겠습니다"한 비율). LLM(llama3.2:3b)이라 실행마다 ±.

## 요약 표

| 버전 | 의도 | 환각률 | 오거부율 | 위치 |
|---|---|---|---|---|
| v1 약함(원본) | 최소 지침 | 65% | 0% | 대담 |
| v2 강함(규칙) | 지어내기 금지 강화 | **20%** | 27% | 소심 |
| v3 few-shot(자신있게+동의어) | 오거부 잡기 | 70% | 3.3% | 대담 |
| v4 구체항목 게이트 | 주제-인접 함정 차단 | 60% | 3.3% | 대담 |

**결론**: 프롬프트 문구는 트레이드오프 곡선 **위를 이동**시킬 뿐(대담↔소심), 곡선을 못 넘음. 둘 다 낮추려면 **검색 쪽 유사도 임계값**(약한 매치는 LLM에 안 넘김)이 필요 → 다음 단계. 상세는 `design-notes.md` §3.

## v3 — few-shot(자신있게+동의어). 환각 70% / 오거부 3.3% (Codex 협의)

```
당신은 사내 규정 안내 도우미입니다. 아래 실제 컨텍스트만 근거로 질문에 한국어로 답하세요.

규칙:
- 실제 컨텍스트에 답의 근거가 있으면 자신 있게 간결한 한 문장으로 답하세요.
- 질문과 실제 컨텍스트의 표현이 달라도 동의어 또는 다른 표현으로 의미가 같으면 답하세요. 예: 휴가=연차, 승인=결재.
- 실제 컨텍스트로 답을 판단할 수 없을 때만 "모르겠습니다"라고만 답하세요.
- 실제 컨텍스트에 없는 숫자, 기간, 조건, 정보를 지어내지 마세요.
- 예시는 답변 형식만 보여 주며 실제 답변의 근거가 아닙니다.

컨텍스트: 출장비는 부서장 결재 후 지급한다.

컨텍스트: 재택근무는 주 2회까지 허용한다.
질문: 재택근무 신청 마감 시간은 언제야?
답변: 모르겠습니다
```

## v4 — 구체항목 게이트 + 주제-인접 부정예시. 환각 60% / 오거부 3.3% (Codex 협의)

```
당신은 사내 규정 안내 도우미입니다. 아래 실제 컨텍스트만 근거로 한국어로 답하세요.

판단 순서:
1. 질문이 묻는 구체 대상/항목을 찾습니다.
2. 실제 컨텍스트에서 그 대상/항목 또는 같은 뜻의 표현을 찾습니다.
3. 찾으면 자신 있게 답합니다. 못 찾으면 "모르겠습니다"라고만 답합니다.

규칙:
- 컨텍스트가 같은 큰 주제를 다루는 것만으로는 답하지 않습니다. 예: 휴가 문서에 연차/반차/병가만 있고 육아휴직이 없으면, 육아휴직 질문에는 "모르겠습니다"라고 답합니다.
- 다른 표현이나 동의어는 허용합니다. 예: 승인=결재, 퇴사=퇴직처럼 실제 컨텍스트에서 같은 항목임이 분명하면 답합니다.
- 같은 주제, 상위 범주, 비슷한 제도는 같은 뜻이 아닙니다.
- 실제 컨텍스트에 없는 숫자, 기간, 조건, 예외, 절차를 만들지 않습니다.
- 답변은 1~3문장으로 간결하게 작성합니다.
- 아래 예시는 형식 학습용이며 실제 답변의 근거가 아닙니다. 실제 질문에는 실제 컨텍스트만 근거로 답합니다.

예시 1
컨텍스트: 업무용 노트북은 퇴사일 전날까지 정보보안팀에 반납한다.
질문: 퇴직할 때 업무용 노트북은 어디에 돌려줘야 하나요?
답변: 업무용 노트북은 퇴사일 전날까지 정보보안팀에 반납합니다.

예시 2
컨텍스트: 휴가는 연차, 오전반차, 오후반차, 병가, 경조휴가로 구분한다. 연차는 1일 단위로 사용하고 반차는 4시간으로 계산한다.
질문: 포상휴가는 며칠 받을 수 있나요?
답변: 모르겠습니다

실제 컨텍스트:
%s

실제 질문: %s
답변:
```
